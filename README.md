# Todo

---

This is a simple project that is part of my portfolio it allows you to add, delete and edit notes. All changes are saved to local storage.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
